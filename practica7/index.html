Práctica 7: PLC

Juan Carlos Guauxochitl Cossío, Andreas Sotomayor Ochoa, Jose Francisco López Hernández y Jorge Hanan Guerrero
Estudiantes de Ingeniería Mecatrónica e Ingnería Automotriz
Fecha: Septiembre 22, 2023
PLC Motores Diagrama escalera Tia Portal

- Resumen -

En esta práctica se llevaron a cabo una serie de tareas utilizando un PLC con el objetivo de utilizar el PLC S71200-1215C para llevar diversos tipos de señales de procesos, y elaborar y enviar acciones de acuerdo con lo que se ha programado en este, el cual se programó en el software de Tia Portal.

- Objetivo General -

Conocer, investigar e identificar el funcionamiento de un PLC.

- Introducción -

En esta práctica se empezó a usar el PLC para realizar las practicas siendo que se hicieron 5 circuitos, con el objetivo de conocer e identificar el uso de un PLC y como acoplarlo a las actividades que anteriormente se han hecho, además de grabar el funcionamiento del PLC con los 5 circuitos.

- Materiales -

Los materiales que se usaron fueron para cada práctica fueron:

• Computadora con software Tia portal y Fluidsim.
• Cables cruzados
• Línea monofásica
• Interruptor electromagnético
• Interruptor tipo pulsador de un solo contacto
• Pulsador con contacto normalmente abierto/cerrado
• Lampará de señalización en general
• PLC

Fig. 1. Circuito materiales

- Desarrollo -

Para la realización de la práctica se han hecho simulaciones en los dos softwares para ver cómo funcionan y se desarrollan antes de implementarlo en física, luego de ello se empezó a desarrollar la parte física donde se conectó el PLC a la computadora, donde se seleccionó el PLC a usar y subió el circuito de escalera para hacer el trabajo.
